Introduction to the Finite Element Method in 1 Dimension

There are plenty of great resources for learning the basis for the Finite Element Method (from here easily shortened to FEM), but this is going to be a simpler interactive introduction to how it works. Inspired by some of the works of Brett Victor I think that FEM can be explained in a much more intuitive manner using interactive visualization. Before we get started though, let's look at some of the excellent and more traditional introductions.

  • For a direct, step-by-step and practical guide check out Zhilin Li of NC State.
  • Francisco–Javier Sayas and his "A gentle introduction to the Finite Element Method" provides an indredible way to understand how FEM works.
  • And most of all I'd like to thank and reccomend Dr. Janet Peterson of FSU for getting me started

This is going to be a simple introduction, as visual possible, intended for those who have some familiarity with calculus and programming who are interested in understanding a way to get started on numerical techniques. After this, feel free to delve deeper into some of those more detailed resources up above. You’re encouraged to follow along, so get out pencil and paper and boot up Octave/MATLAB.

The Problem

For this tutorial we'll be considering one dimensional boundary value problems of the form \(y'' + p(x)y = f(x)\). Specifically the problem

\(y'' + 4y = 0\)
\(y(0)=-2 \;\;\;\; y(\frac {\pi} {4})=10\)

If you need a bit of a refresher on differential equations go check it out. Since this is a relativly simple problem we can find the exact solution analytically. Because of that we'll be able to check our work and do a brief analysis on how well the approximation works before applying FEM to more complex problems. The exact answer is:

\(y(x) = -2cos(2x) + 10sin(2x)\)
